# TicketBot
Discord Crypto Display Bot. Developed in Python using discord.py



## Preview

## Installation guide

```
$ git clone https://github.com/itsmat/CryptoDisplayBot.git or download
$ python -m pip install -r requirements.txt
Edit config.py
$ python main.py
```
